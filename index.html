QuantumMC+

A Modern GUI Minecraft Utility Client for Fabric 1.21.x

Experience next-generation gameplay with advanced combat, movement, and rendering modules. Based on Oyvey Ported, reimagined for modern Minecraft.

Download Now View on GitHub

Powerful Features

Everything you need for an enhanced Minecraft experience

⚔️ Combat

  • Aura - Auto-attack nearest entity with filtering
  • AutoTotem - Auto-equip totem at low health
  • Criticals - Guarantee critical hits
  • KeyPearl - Throw ender pearls instantly

🏃 Movement

  • Speed - Strafe-based speed multiplier
  • Step - Auto-step up blocks
  • Sprint - Multi-directional sprinting
  • SafeWalk - Prevents edge falls

👁️ Render

  • Fullbright - Increase world brightness
  • ESP - Entity highlighting system
  • Tracers - Lines toward nearby entities
  • BlockHighlight - Wireframe overlays

👤 Player

  • AutoTool - Auto-switch to best tool
  • AutoEat - Eat when hungry
  • AutoRepair - Durability warnings
  • NoFall - Cancel fall damage

📊 HUD

  • Coordinates - XYZ with nether conversion
  • FPS Counter - Performance monitoring
  • Status Display - Health & hunger
  • Watermark - Client branding overlay

⚙️ Client

  • Notifications - Module toggles
  • Overlay Toasts - On-screen alerts
  • MCF - Friends system
  • ChatFilter - Message filtering

Installation Guide

Get QuantumMC+ running in minutes

1

Install Minecraft 1.21.11

Download and install Minecraft Java Edition 1.21.11 from the official launcher.

2

Install Fabric Loader 0.18.4

Download Fabric Loader from fabricmc.net and run the installer for 1.21.11.

3

Download QuantumMC+ JAR

Get the latest JAR from Releases page.

4

Place in Mods Folder

Move the JAR file into your .minecraft/mods/ folder.

5

(Optional) Fabric API

Install Fabric API 0.141.1+1.21.11 for enhanced compatibility.

6

Launch & Enjoy

Select Fabric profile in launcher and start playing with QuantumMC+!

Default Keybinds

Get familiar with essential controls

Key Action
Right Shift Open ClickGUI
Right Control Open HUD Editor
. (Period) Command Prefix

💡 Configure custom keybinds per-module via ClickGUI or use the .bind <module> <key> command.

Essential Commands

Master the command system

.bind <module> <key>

Bind a key to any module

.toggle <module>

Enable or disable a module

.friend <add/remove/list/clear>

Manage your friends list

.drawn <module>

Show/hide in array list

.prefix <char>

Change command prefix

.profile [list/set/delete]

Manage configuration profiles

.help [page]

List all available commands

.<module>

Configure module settings
